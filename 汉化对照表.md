## 四、提示文本（Tooltip）

| 参数 | 原文 | 中文 |
|---|---|---|
| `face` | `Select a face where dovetails will be placed` | `选择要放置燕尾榫的面` |
| `edge` | `Select an edge on the face where dovetails will be placed` | `在选定面上选择要放置燕尾榫的边` |
| `edge_length` | `Usually it's set to board width parameter like "drawer_height".` | `通常设置为板材宽度参数，例如 "drawer_height"。` |
| `angle` | `Angle of the dovetails side. For 1:8 - 7 deg. For 1:6 - 9.5 deg.` | `燕尾榫侧边的角度。1:8 比例对应 7°，1:6 比例对应 9.5°。` |
| `height` | `Height of the dovetails from the board edge. Usually it's set to other board thickness parameter like "drawer_side_thickness".` | `燕尾榫从板边起始的高度。通常设置为另一块板的厚度参数，例如 "drawer_side_thickness"。` |
| `thickness` | `Thickness of the dovetails. Usually it's set to board thickness parameter like "drawer_front_thickness".` | `燕尾榫的厚度。通常设置为板材厚度参数，例如 "drawer_front_thickness"。` |
| `ratio` | `Ratio of the maximum width of the tails to the maximum width of the pin.` | `榫头最大宽度与榫肩最大宽度的比值。` |
| `pin` | `Maximum width of the full pin between dovetails. Increase this to produce more bigger dovetails, decrease otherwise.` | `燕尾榫之间完整榫肩的最大宽度。增大此值可生成更大的燕尾榫，减小则反之。` |
| `start_type` | `Determine whenever to start with full pin or half pin.` | `指定以完整榫肩还是半榫肩开始。` |
| `params_prefix` | `All generated dovetails has their parameters added as user parameters with the names prefixed with this text. Usually it's set to name of the part like "drawer_front_dovetails_"` | `所有生成的燕尾榫参数将作为用户参数添加，其名称以此文本为前缀。通常设置为零件名称，例如 "drawer_front_dovetails_"` |
